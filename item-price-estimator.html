Item Price Estimator

✦

The Item Price Estimator app was commissioned for personal use; the client's request was to be able to have an easy way to price objects they were planning to sell. Special emphasis was placed on using retrieval augmented generation (RAG) to retrieve search results from the internet containing the same (or similar) object in mind, evaluate the object's price based on quality, and store the results in a table that would be saved on the user's local system.

The app first requests access to the device's camera through a MediaDevices API built into the browser itself. Once the picture is taken, a pre-trained AI image classification model called MobileNet identifies the object (this runs entirely in-browser thanks to Tensorflow); then the results are sent to the backend, where an API key calls Google Gemini to do a grounding search for the real-world price.

Whether you're organizing a garage sale, getting rid of old furniture, or just satisfying your curiosity, the Item Estimator provides a simple way to help you out.
