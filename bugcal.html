🪲 BugCal

✦

BugCal is a bug-themed personal calendar created to demonstrate Claude AI's ability to implement multiple AI agents working together. These agents detect scheduling conflicts, verify federal holidays via live web search, and allow the user to schedule events by writing down the specified day(s), time(s), and month(s) they're set on. The specifics of when an event occurs, if it is reoccuring, and how long it lasts are fully customizable.

Color-coordinated sorting, filtering, timezone management, voice command support, and event reminders are all supported.
