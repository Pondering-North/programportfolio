BugCal

✦

This is a simple calendar app that I created to test Claude AI's creation and implementation of multiple ai agents. BugCal uses these agents to parse natural language event descriptions, detect scheduling conflicts, and verify US federal holidays via live web search. All of this wrapped in a theme centered around insects. It's not a very complex app, but it's cute and was a good test of Claude's skills

The app is similar to many apps you can find online. You can add events and specify which days or times they're on, how often they reoccur, and even what color and bug-themed icons they have. Events are saved to your browser's local storage so that you can access them at any time.
