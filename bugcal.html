🪲 BugCal

✦

BugCal is a bug-themed personal calendar that I created to demonstrate Claude AI's ability to create and implement multiple AI agents working together. The app uses these agents to parse natural language event descriptions, detect scheduling conflicts, and verify US federal holidays via live web search.

Features

  • Natural Language Parsing: Enter events like "standup every Mon/Wed/Fri at 9am" and the AI agent parses the details automatically
  • Recurring Events: Support for daily, weekly, biweekly, monthly, yearly, or custom weekday patterns
  • Flexible Repeat Endings: Events can repeat infinitely, until a specific date, or for a set number of occurrences
  • Conflict Detection: AI agent warns you when new events overlap with existing ones
  • Holiday Verification: AI agent verifies US federal holiday dates using live web search
  • Persistent Storage: Events are saved to your browser's local storage and persist across sessions
  • Bug-Themed Icons: Each event can be customized with adorable bug icons and colors

Planned Features

Future updates will include day/week/month/agenda views, event reminders, search & filtering, timezone management, and voice command support.
